# 서울 아파트 실거래가 기반 가격 예측 프로젝트

데이터마이닝 A분반 2팀 · 학기말 프로젝트

서울 아파트 매매 실거래가에 입지(지하철·학교)·건물 특성·거시경제(기준금리) 변수를 결합하여
가격 결정 요인을 분석하고, 머신러닝 회귀 모델로 거래금액을 예측한다.

## 분석 개요

- **분석 기간**: 2023년 1월 ~ 2025년 12월 (36개월)
- **데이터 규모**: 142,463건 × 33개 변수 (전처리 후, 결측치 0건)
- **목표**: ① 가격 결정 요인 분석 ② 거래금액 예측 ③ 유사 아파트 추천

## 저장소 구조

```

docs/
  데이터수집_담당파트.md                # 데이터 수집·변수 생성 상세 문서
```

## 변수 생성 요약

- **지오코딩**: 단지명+도로명 기준 고유 단지만 추출해 VWorld API로 좌표 변환 (API 호출 약 16배 절감)
- **입지 변수**: haversine 공식으로 최근접 지하철역·학교까지 거리, 반경 1km 내 학교 수 계산
- **파생 변수**: 전용면적(평), 평형대, 건물연식, 구별/단지별 평균 평당가
- **거시 변수**: 계약년월 기준 기준금리 결합

## 모델링 결과

5개 scikit-learn 회귀 모델 비교 (타깃: 거래금액 log 변환):

| 모델 | RMSE(만원) | R² |
|---|---:|---:|
| 선형 회귀 | 45,012 | 0.8806 |
| K-최근접 이웃 | 18,014 | 0.9605 |
| 랜덤 포레스트 | 12,994 | 0.9788 |
| 그래디언트 부스팅 | 23,212 | 0.9414 |
| **히스토그램 기반 부스팅** (채택) | 18,511 | 0.9647 |

- **최종 예측 모델**: 히스토그램 기반 그래디언트 부스팅 — 속도·성능 균형 고려
- **최종 입력 변수 6개**: 단지별 평균평당가, 전용면적, 구별 평균평당가, 기준금리, 건물연식, 최근접 역 거리
- **최종 성능**: MAE 10,173만원 / RMSE 18,971만원
- **유사 아파트 추천**: NearestNeighbors로 입력 조건과 가장 가까운 5개 단지 제시
